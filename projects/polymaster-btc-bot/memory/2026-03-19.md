### 1. FastRequoteEngine Dependency Injection ✅
**Problem**: Constructor rejected `market` parameter, required `polly_client` and `signer`

**Solution**: Implemented proper dependency injection pattern

```python
# Before (failed):
engine = FastRequoteEngine(market='BTC-USD')  # ❌ TypeError

# After (works):
engine = FastRequoteEngine(polly_client=client, signer=signer)  # ✅
```

### 2. Bidirectional Quote Implementation ✅
**Problem**: `generate_bidirectional_quote()` method was missing from strategy

**Solution**: Added method that uses BlackScholesPricer infrastructure to generate bid/ask quotes with configurable spread_bps

**Key Features**:
- Uses last price or calculated fair value as base
- Applies half_spread on each side (bid: -spread/2, ask: +spread/2)
- Returns dict with full quote metadata
- Handles edge cases (price_history < 3, no pricer)
