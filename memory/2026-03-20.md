## 🔑 GitHub PAT Token Solution (Fri Mar 20, 2026 12:41 AM PDT)

**时间**: Friday, March 20th, 2026 — 12:41 AM PDT  
**问题**: "Invalid username or token. Password authentication is not supported for Git operations."

---

### ✅ **问题根源:**

GitHub 已**禁用账号密码 Git 认证**。必须使用：
1. **Personal Access Token (PAT)** ← 推荐，最简单！
2. SSH Key (如果已配置)

---

### 🎯 **快速解决方案：生成 PAT Token**

#### **Step 1: 在浏览器访问**
```
https://github.com/settings/tokens/new
```

#### **Step 2: 填写表单**
```
Token name: polymaster-btc-bot
Note: For Git push to polymaster-btc-bot repo
Expiration: No expiration (或 90 days)
Select scopes: 
    [✓] repo - Full control of private repositories
Click "Generate token"
```

#### **Step 3: 复制 Token!**
长这样：`github_pat_XXXXXXXXXX...`  
⚠️ **保存好！只会显示一次！**

---

#### **Step 4: 在 VPS 运行克隆命令**

替换 `<YOUR_PAT_TOKEN>` 为你的实际 token:

```bash
# 清理失败的尝试
cd /root
rm -rf polymaster-btc-bot

# 用 PAT token 克隆
git clone https://<TOKEN>@github.com/stevenchn56/polymaster-btc-bot.git

# 例如:
# git clone https://ghp_xxxxxxxxxxxxxxxxxxxxxxxx@github.com/stevenchn56/polymaster-btc-bot.git
```
